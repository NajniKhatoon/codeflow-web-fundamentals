## 📚 Weekly Breakdown

### Week 1 – Basic HTML Structure
- Semantic HTML elements (`header`, `main`, `section`, `footer`)
- Text formatting (`strong`, `em`, `code`, `sup`)
- Basic CSS styling: fonts, colours, card layout
- Personal introduction and module expectations

### Week 2 – Lists & Tables
- Ordered and unordered lists
- HTML tables with `rowspan` and `colspan`
- Game results table and weekly timetable
- CSS class-based styling for table rows

### Week 3 – Links & Images
- Internal, external, and `mailto:` links
- CSS pseudo-classes: `:link`, `:visited`, `:hover`, `:active`
- Responsive images with hover effects
- Section navigation with anchor links

### Week 4 – Forms & CSS Layout
- HTML form elements: `text`, `email`, `tel`, `select`, `textarea`, `radio`, `checkbox`
- `fieldset` and `legend` for form grouping
- CSS float layout with `clear`
- Focus and hover states for form inputs

### Week 5 – SVG Graphics
- Inline SVG shapes: `circle`, `polygon`, `rect`, `line`
- SVG grouping with `<g>` and semantic attributes (`role`, `title`, `desc`)
- CSS hover transitions on SVG elements
- Interactive cat illustration with colour-changing eyes on hover

### Lab – BrightWave Creative Agency
- Multi-section business webpage
- Campaign portfolio table with `rowspan` and `tfoot`
- Full project enquiry form with validation
- Structured navigation with anchor links
